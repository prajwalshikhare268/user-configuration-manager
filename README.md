# User Configuration Manager

A simple Python project that allows users to manage application settings.

## Features

* Add new settings
* Update existing settings
* Delete settings
* View all settings

## Technologies Used

* Python

## Functions

* add_setting(settings, setting)
* update_setting(settings, setting)
* delete_setting(settings, key)
* view_settings(settings)

## Example Settings

* Theme: Dark
* Notifications: Enabled
* Volume: High
